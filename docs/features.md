# Features

## User Authentication
- Sign up, log in, and log out functionality.
- Password recovery and profile management.

## Task Creation and Management
- Create new tasks with details such as title, description, due date, and priority.
- Edit, delete, and mark tasks as complete.

## Project Management
- Create and manage projects.
- Assign tasks to specific projects and view project progress.

## Task Prioritization
- Set priority levels (high, medium, low) for tasks.
- Filter tasks by priority.

## Due Dates and Reminders
- Set due dates for tasks.
- Notifications for upcoming deadlines.

## Search and Filter
- Search for specific tasks.
- Filter tasks by status, priority, due date, or project.
